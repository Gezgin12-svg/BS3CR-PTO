# UdemigVerse - Kripto Para Platformu

## Neler Öğrendim, Neler Kullandım

---

### 🧠 Öğrendiklerim

- Kripto para piyasası temel kavramları (Bitcoin, Ethereum vb.)
- Kullanıcı dostu arayüz (UI) tasarımı
- Responsive web tasarımı prensipleri
- Slider / Carousel bileşeni oluşturma
- Navigasyon menüsü yapılandırması
- Dark tema ile modern web tasarımı

---

### 🛠️ Kullandığım Teknolojiler

#### Frontend

- **HTML5** – Sayfa yapısı ve semantic elementler
- **CSS3** – Stil, animasyon ve dark tema tasarımı
- **JavaScript** – Dinamik içerik ve slider işlevselliği
- **Bootstrap**

#### Tasarım & UI

- **Responsive Design** – Farklı ekran boyutlarına uyum
- **Dark Theme** – Modern kripto platformu görünümü
- **Animasyonlu Hero Section** – Kripto para görselleriyle slider

#### İçerik & Özellikler

- 🪙 Kripto para listeleme (Popular Cryptocurrencies)
- 📰 Son Haberler bölümü
- 💱 Crypto Sat butonu
- 🔗 Navigasyon: Anasayfa, Kripto, Özellikler, Yorumlar, Haberler, İletişim
